# 📚 도서관 웹 사이트 제작 프로젝트 (Library Web Project)

Java Servlet & JSP 기반의 도서관 사이트입니다.  
게시판 중심 기능을 포함하여 웹 애플리케이션의 주요 기능들을 직접 설계하고 구현하였습니다.

---

## 📝 프로젝트 개요

- **프로젝트명**: 도서관 웹 사이트  
- **개발 기간**: 2025.04.23 ~ 2025.05.23 (1개월)  
- **참여 인원**: 4명
